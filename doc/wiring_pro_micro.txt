*** Назначение выводов Arduino Pro Micro

TXO HWON - Управление питанием периферии -> к плате питания
RXI Кнопка 2 (выбор)
  2 SDA ---+--> к RTC и датчику давления
  3 SCL --/
  4 Подсветка дисплея -> к LGT дисплея
  5 Красный светодиод
  6 Buzzer (пьезодинамик)
  7 Вход прерываний -> к RTC
  8 Вход прерываний 2 для RTC PCF8563
  9 Зелёный светодиод
 RX SPI SS -> контакт CE дисплея
 TX SPI D/C  -> Контакт DC дисплея 5110
 10 Синий светодиод
 16 SPI MOSI -> контакт DIN дисплея
 14 SPI MISO -> не используется
 15 SPI CLK -> контакт CLK дисплея
 A0 Сенсор заряда батареи -> к плате питания
 A1 Кнопка 3 (вниз)  \
 A2                   -> к кнопкам
 A3 Кнопка 1 (вверх) /
 J1 Заряд батареи -> к плате питания

RAW При расширенном управлении энергосбережением - не используется.
    При стандартном - [+] батареи и питание часов RTC
    
RST Кнопка сброса -> к кнопке
VCC При расширенном управлении энергосбережением - вход питания -> к плате питания
    При стандартном - выход 3.3в -> питание дисплея и датчика давления

Между выходами Arduino и светодиодами включаются ограничительные резисторы сопротивлением
330ом - 1кОм. Обычно в цепи зелёного светодиода сопротивление нужно брать несколько больше,
чем для остальных. Общий вывод светодиода подключается к VCC или общему проводу для светодиодов
с общим анодом и общим катодом соответственно.

Второй вывод всех кнопок соединяется с общим проводом.

Пьезодинамик подключается к контакту 5 через резистор 51-200 ом. Второй вывод пьезодинамика
соединяется с общим проводом.


*** Доработка контроллера Ardiono Pro Micro

С платы Arduino удаляются следующие компоненты:
- Все светодиоды
- 1к (102) резисторы рядом со светодиодами
- Защитный диод рядом с USB разъёмом (на рисунке - с маркировкой S4)

Если применяется расширенное управление энергосбережением, то дополнительно удаляется предохранитель
(расположен под диодом) и стабилизатор 3.3в (расположен под предохранителем). Стабилизатор удаляйте
аккуратно, т.к. он понадобится для схемы расширенного управления энергосбережением.

К выводам RX и TX вместо удалённых резисторов подпаиваются тонкие проволочки. Подпаивать лучше к обеим
площадкам, где раньше был резистор - для прочности. Оставляем "хвостики" длиной около 5мм. Такая же
проволочка припаивается к ближней к USB разъёму площадке перемычки J1. Защищаем пайку каплей
эпоксидной смолы.


*** Подключение микросхемы часов реального времени (RTC)

Микросхема часов распаивается поверх Arduino либо поверх датчика давления:

           +---------+
 SDA <-----+5       4+---*------------+
           |         |   |            |
 SCL <-----+ PCF8583 +---+            |
           |         |                |
   7 <-----+         +------------+   |
           |         |            |   |
       +---+8       1+--*----o|o--+   |
       |   +----u----+  |   кварц     |
       |        C1 22pF |   32768     |
       *----------||----+             |
       *-----|<|----------------------+----< (+) основной батареи
       |                              |
       *-----|<|------[+ ### -]-------*----|
       |            батарея 1.5в      |
       |                              |
       +----------------||------------+
                    C2 0.22мкф



           +---------+
 SDA <--+--+5       4+----------------+
           |         |                |
 SCL <--+--+ PCF8563 +---> 8          |
           |         |                |
   7 <-----+         +------------+   |
           |         |            |   |
       +---+8       1+--*----o|o--+   |
       |   +----u----+  |   кварц     |
       |        C1 22pF |   32768     |
       *----------||----+             |
       *-----|<|----------------------+----< (+) основной батареи
       |                              |
       *-----|<|------[+ ### -]-------*----|
       |            батарея 1.5в      |
       |                              |
       +----------------||------------+
                    C2 0.22мкф

Конденсатор C1 используется для подстройки хода часов. Можно обойтись без него, но уход часов
при этом будет порядка 30с в сутки. C2 - любой керамический 0.22мкф и выше. Диоды - диоды Шоттки
либо кремниевые, в последнем случае напряжение батареи увеличить до 3в.
